Параметры:
1) Директория конфигурации, содержащая файлы соответственно config.txt.
2) Директория данных, содержащая директории каждого из доменов.
3) uid пользователя, под которым будет работать сервер
4) gid пользователя, под которым будет работать сервер (у пользователя с uid=uid должен быть такой gid)

Директории должны быть заданы в виде /{dir_1}/.../{dir_n}. (!)
